1. Чтобы создать git репозиторий из локального каталога на своем компьютере, нужно выполнить следующую последовательность команд в терминале или консоли:


cd /path/to/local/folder   # переход в локальный каталог
git init                   # инициализация git
git add .                  # добавление всех файлов в stage area
git commit -m "Initial commit"   # фиксация изменений


2. Чтобы отправить локальный репозиторий на GitHub, нужно выполнить следующие шаги:

- Создать пустой репозиторий на GitHub
- Связать локальный репозиторий с репозиторием на GitHub с помощью команды git remote:

git remote add origin https://github.com/username/repo-name.git


- Отправить изменения на сервер GitHub с помощью команды git push:


git push -u origin master


3. Чтобы получить изменения с GitHub на локальный компьютер, нужно выполнить команду git pull:


git pull origin master


4. Команда git status показывает состояние вашей рабочей копии git, т.е. какие файлы были изменены, добавлены или удалены.

Команда git log показывает историю коммитов в вашем репозитории.

Команда git remote -v показывает список удаленных репозиториев, связанных с вашим локальным репозиторием, и какие URL-адреса они имеют.
